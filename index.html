Developed Bioinformatics Tools & Platforms 生信工具与平台开发

Hastat (Gene Haplotype Analysis Tool)

An independently developed modular toolkit for comprehensive gene haplotype analysis and visualization in natural populations. It comprises four core modules: View (efficient genotype processing via gffutils and pysam, calculating diversity indices like π and FST), Stat (rigorous phenotypic association via ANOVA/Tukey's HSD), Network (MST/MSN evolutionary trajectory construction), and Plot (publication-quality, highly customizable visualization integrating gene structures using TOML configurations).

基于 Python 自主开发的模块化基因单倍型分析与可视化工具。包含四大核心模块：数据视图 (高效处理变异数据及计算 π、FST 等遗传统计量)、统计分析 (基于 ANOVA 与 Tukey's HSD 的表型变异严格统计评估)、网络构建 (MST/MSN 进化网状关系解析) 以及可视化 (结合 TOML 配置实现基因结构与单倍型数据的深度整合与出版级高度定制化绘图)。

https://github.com/swu1019lab/hastat
GWASNexus (Post-GWAS Analysis Platform)

A comprehensive analysis platform designed to facilitate system-wide genetic analysis in crops. It integrates GWAS scanning, dynamic LD-based clustering (replacing fixed physical distances), epistasis detection, and genetic network construction for pleiotropy. GWASNexus automatically aggregates outputs into a unified, interactive HTML dashboard.

一款专为作物系统级遗传分析设计的全面 Post-GWAS 分析平台。集成了 GWAS 扫描、基于动态 LD 的聚类算法、上位性效应检测以及揭示多效性的遗传网络构建功能，并将所有输出结果自动汇聚成统一的交互式 HTML 报告。

https://github.com/swu1019lab/GWASNexus
BnaPan50T (Target Capture Sequencing Platform)

A 50K liquid chip (Target Capture Sequencing) SNP genotyping platform developed for rapeseed. It contains 54,765 core SNPs (including pan-genomic and trait-associated markers). Validated in a DH population, it achieved 60% target region specificity, 78% coverage uniformity, and 93% genotyping accuracy. It has been successfully applied to QTL mapping and DNA fingerprinting, providing a highly accurate and cost-effective molecular marker resource for genomic breeding.

为油菜开发的一款 50K SNP 靶向捕获测序 (TCS) 液相芯片平台。平台包含 54,765 个核心 SNP 标记。在 DH 群体测试中，该芯片展现出 60% 的目标区域特异性、78% 的覆盖均一性以及 93% 的基因分型准确性。该平台具备高精准度与高性价比，成功应用于 QTL 定位与 DNA 指纹图谱构建，为油菜遗传分析与基因组育种提供了宝贵的分子资源。

qPrimerDB 2.0

A comprehensive gene-specific qPCR primer database for 1172 organisms, expanded to contain 430 million primer pairs covering 43 million genes. To support genome-scale primer design, we developed the Python-based qPrimer tool for high-throughput, highly specific design via parallel computing. The platform offers a complete online system for primer search, design, evaluation, and submission, significantly accelerating gene expression research.

涵盖 1172 种生物的基因特异性 qPCR 引物综合数据库。平台大幅扩展，收录了 4.3 亿个引物对（覆盖 4300 万个基因）。为满足基因组规模需求，自主开发了基于 Python 的高通量 qPrimer 工具，通过并行计算及特异性优化显著提升设计效率。平台提供完整的在线引物搜索、设计、检测和标注系统，极大简化了基因表达分析流程。

https://qprimerdb.biodb.org/   |   qPrimer Tool
PGFAT (Pan-Gene Family Analysis Toolkit)

A Python-based command-line pipeline designed to handle complex multi-genome datasets. It automates workflows from HMM-based member identification to pan-genomic classification, featuring a dual-strategy phenotype association module and an advanced visualization module.

基于 Python 开发的高通量命令行分析流程。自动化了从基于 HMM 的成员鉴定、系统发育分析到泛基因组分类的全流程。内置了双策略表型关联模块，并集成了可自动生成高质量矢量图的可视化模块。

https://github.com/swu1019lab/PGFAT
BrassicaIBD (Intelligent Breeding Platform for Rapeseed)

An intelligent breeding platform for rapeseed based on B/S architecture. It integrates three core modules: germplasm query, gene mining, and breeding design (iTrait, iArray, and iGS), achieving one-stop core gene mining and targeted breeding design.

基于 B/S 架构开发的油菜智慧育种平台。集成了种质查询、基因挖掘（内嵌多种在线工具）和育种设计（iTrait, iArray, iGS）三个核心模块，实现了海量种质资源管理、核心基因一站式挖掘及靶向育种设计。

https://brassicaibd.cn/
